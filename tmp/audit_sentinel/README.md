# 🔍 Sentinel Research Infiltrator (v1.0.0)

## Part of the Digital Swarm Elite Asset Collection

### Overview

The **Sentinel Research Infiltrator** is an autonomous AI agent designed for deep-web intelligence gathering. Unlike standard LLMs, this agent proactively searches the surface and deep web to synthesize structured reports on any given target.

### Elite Features

- **Deep Scrape Technology**: Uses Firecrawl-powered autonomous navigation to bypass surface-level results.
- **Dual-Agent Architecture**:
  - **The Researcher**: Gathers and filters raw technical data.
  - **The Elucidator**: Polishes, structures, and synthesizes the final intelligence report.
- **Cyberpunk UI**: Streamlit-powered dashboard for real-time monitoring of the research process.

### Installation Instructions

1. **Clone/Unzip** this asset into your local environment.
2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Configure your `.env` file with your **OpenAI API Key** and **Firecrawl API Key**.
4. Deploy the Sentinel:

   ```bash
   streamlit run sentinel_researcher.py
   ```

### Technical Specs

- **Language**: Python 3.9+
- **Frameworks**: OpenAI Agents SDK, Streamlit, Firecrawl.
- **Protocol**: Swarm Infiltration v1.
