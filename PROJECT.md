# Digital Swarm: Planet ONO eCommerce

## Vision
A high-performance, "Planet ONO" brutalist industrial eCommerce platform. Focus on raw aesthetics, sharp typography, and seamless checkout.

## Architecture
- **Framework**: Next.js 15 (App Router)
- **Styling**: Vanilla CSS (CSS Modules)
- **Design System**: Brutalist (Planet ONO)
- **Database/Auth**: Supabase + Clerk

## Core Features
1. **Product Catalog**: Dynamic product grid with brutalist layout.
2. **Cart Management**: Persistent cart with industrial-styled UI.
3. **Checkout Flow**: Secure payment integration.
4. **Admin Dashboard**: Comprehensive order and product management.
