# 💰 Swarm Finance Infiltrator (v1.0.0)

## Advanced Intelligence Tier - xAI Oracle

### Overview

The **Swarm Finance Infiltrator** is a high-speed technical oracle designed for real-time market infiltration. Powered by **xAI's Grok** and specialized financial tools, it synthesizes market data, technical indicators, and analyst sentiments into actionable trading intelligence.

### Elite Features

- **xAI Grok Integration**: Leverages the raw intelligence of xAI's flagship models for market reasoning.
- **Real-Time Data Uplink**: Integrated with `yfinance` for live stock, crypto, and currency metrics.
- **Technical Analysis Engine**: Automatically calculates RSI, MACD, and other technical signals from live data packs.
- **Sentimental Scrape**: Uses DuckDuckGo search logic to gauge market mood from news and social signals.
- **Cyberpunk Command Line Interface**: Optimized for terminal-based high-speed data extraction.

### Installation Instructions

1. **Unzip** the asset.
2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Configure your `.env` file with your **XAI_API_KEY**.
4. Infiltrate the markets:

   ```bash
   python src/agents/finance_agent.py
   ```
