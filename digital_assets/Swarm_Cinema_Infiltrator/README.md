# 🎬 Swarm Cinema Infiltrator (v1.0.0)

## God Tier Oracle - Multi-Agent Film Intelligence

### Overview

The **Swarm Cinema Infiltrator** is a professional-grade multi-agent engine designed to materialize movie concepts from raw abstract signals. It coordinates an elite team of Script Architects and Talent Oracles to generate comprehensive production blueprints.

### Elite Features

- **ScriptArchitect Agent**: Expert screenplay engineer that builds 3-act structures, complex character webs, and plot-twist matrices.
- **TalentOracle Agent**: Integrated with SerpApi to perform live talent scouting and casting validation based on real-world actor availability and performances.
- **Production Synthesis**: The MovieProducer team orchestrates the entire flow, delivering a unified concept overview ready for a pitch or development.
- **Interactive Command Center**: Streamlit-powered dashboard with cyberpunk aesthetics for real-time concept development.

### Installation Instructions

1. **Unzip** the asset.
2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Configure your `.env` file with your **GOOGLE_API_KEY** (Gemini) and **SERP_API_KEY**.
4. Launch the Cinema Hive:

   ```bash
   streamlit run agent.py
   ```
