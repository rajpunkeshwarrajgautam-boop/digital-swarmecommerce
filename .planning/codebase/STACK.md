# Tech Stack: Digital Swarm

## Core Frameworks
- **Frontend**: Next.js 15 (App Router)
- **Runtime**: Node.js 20+
- **Language**: TypeScript 5.7+

## UI & Styling
- **CSS**: Vanilla CSS (Tailwind CSS 4.0 used for utility classes)
- **Animations**: Framer Motion, GSAP
- **Icons**: Lucide React, Tabler Icons

## Backend & Infrastructure
- **Database**: Supabase (PostgreSQL)
- **Auth**: Clerk
- **State Management**: Zustand
- **Payments**: Cashfree PG, Stripe, Razorpay (Multi-gateway support)
- **API Clients**: native `fetch` with `Zod` validation
- **Emails**: Resend

## Deployment
- **Platform**: Vercel
- **CI/CD**: GitHub Actions (planned)
