# Phase 4.3: Performance optimization - Plan

This plan focuses on front-end delivery speed to ensure a ultra-fast "Digital Swarm" experience.

## Tasks

### 1. Image Infrastructure [next.config.ts]
- [ ] Enable `formats: ['image/avif', 'image/webp']` in `images` config.
- [ ] Ensure all product images use `next/image`.

### 2. Google Fonts Optimization [src/app/layout.tsx]
- [ ] Replace custom CSS font-face with `next/font/google`.
- [ ] Apply `Outfit` to headings and `Inter` to body text.
- [ ] Use CSS variables for font mapping.

### 3. LCP Priority [src/app/page.tsx, src/app/product/[slug]/page.tsx]
- [ ] Add `priority` prop to hero images.
- [ ] Ensure `sizes` attribute is correctly calculated for mobile-first delivery.

### 4. Skeletal Loading [src/components/ui/Skeleton.tsx]
- [ ] Build a reusable, industrial-themed `Skeleton` component with a subtle pulse animation.
- [ ] Implement `ProductGridSkeleton` for use in `Suspense` boundaries.

### 5. Script Audit [src/app/layout.tsx]
- [ ] Defer non-critical analytics (Facebook Pixel) to ensure it doesn't block the main thread during hydration.

## Verification Checklist

- [ ] LCP < 1.0s (Manual/Lab check).
- [ ] No CLS from font loading.
- [ ] Images served as AVIF where supported.
- [ ] Product grid remains fluid during loading.
