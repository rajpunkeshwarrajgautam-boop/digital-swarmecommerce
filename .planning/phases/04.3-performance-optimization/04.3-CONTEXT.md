<decisions>
## Implementation Decisions

### Image Delivery
- **Choice**: Modern formats (AVIF first, then WebP) with explicit priority for hero/LCP images.
- **Rationale**: AVIF provides up to 30% further compression over WebP.

### Font Optimization
- **Choice**: `next/font/google` for Inter and Outfit.
- **Action**: Use `display: swap` and subsetting to ensure zero layout shift.

### Component Hydration
- **Decision**: Selective hydration for non-critical interactive components (e.g., chat, affiliate banners).
- **Loading**: Standardize on skeletal loaders for dynamic content blocks.

### the agent's Discretion
- Standard next/image sizing conventions.
- Defer non-critical analytics/Clerk hooks where possible without breaking auth.

</decisions>
