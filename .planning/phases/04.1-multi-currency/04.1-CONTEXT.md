# Phase 4.1: Multi-currency support - Context

**Gathered:** 2026-03-30
**Status:** Ready for planning
**Mode:** Smart Discuss (Autonomous Phase)

<domain>
## Phase Boundary

Implement multi-currency support throughout the store.
- Users view prices in their select currency.
- Product listings and product detail pages show dynamic prices.
- Cart and Checkout reflect the chosen currency.
- Cashfree orders are created with the correct ISO currency code.

</domain>

<decisions>
## Implementation Decisions

### Currency Provider
- **Choice**: Static conversion table.
- **Rationale**: Simplifies initial implementation without external API dependency/latency.

### Supported Currencies
- **INR** (Base)
- **USD**
- **EUR**
- **GBP**

### Price Format
- **Format**: [Symbol] [Amount] (e.g., $ 99.00, ₹ 7450.00)

### the agent's Discretion
- Standard HSL industrial styling for the currency switcher.
- Use `localStorage` for persisting currency selection.

</decisions>
