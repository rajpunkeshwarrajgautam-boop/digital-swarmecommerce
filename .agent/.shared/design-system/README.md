# 🎨 Design System Module (AntiGravity IDE)

**Status**: Core Capability
**Type**: Shared Module (Components & Specs)

The Design System module ensures visual and functional consistency across all AntiGravity-derived interfaces.

## 💎 Principles

1. **Fluidity**: All interactive elements must have smooth 200-400ms transitions.
2. **Contrast**: High legibility using HSL-based color tokens.
3. **Intentionality**: Use micro-interactions (hover, scales) to guide user attention.

## 🛠️ Standards

- **Font**: Inter, Outfit, or Poppins (SANS-SERIF).
- **Theme**: Dark Mode by default for Digital Swarm.
- **Borders**: Subtle `1px border-white/10` with `backdrop-blur`.
