---
description: Initialize a new GSD-spec project.
---

1. **Questionnaire**: Ask the user for project description, main features, and target tech stack. 
2. **Setup**: Create `PROJECT.md`, `REQUIREMENTS.md`, and `ROADMAP.md` templates.
3. **Draft Requirements**: Generate initial functional and non-functional requirements based on the discussion.
4. **Draft Roadmap**: Break the project into 3-5 verifiable milestones.
5. **Freeze State**: Initialize `STATE.md` with "Project Start."
